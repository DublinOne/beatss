# Prodigy Beast Sequencer 🎧⚡

A high-performance, AI-powered web sequencer inspired by the iconic sounds of **The Prodigy** and 90s underground warehouse raves.

## 🚀 Key Features

### 1. ⚡ Live Performance Pads
Real-time audio manipulation tools for live sets:
- **STUTTER**: Rapidly repeat the current step for build-ups.
- **TAPE STOP**: Authentic vinyl stop effect with volume/filter ramping.
- **REVERSE**: Instantly flip your patterns backward.
- **GLITCH**: High-energy randomization of Distortion and Filter FX.

### 2. 🎤 Gritty Vocal Slicer
A unique 16-step pitch slicer for vocal stabs:
- Draw pitch curves using 16 mini-vertical sliders (-12 to +12 semitones).
- Toggle specific steps using trigger buttons.
- Synthesized "EY!" engine or upload your own samples.

### 3. 🙌 Crowd Hype AI
Simulate a massive festival atmosphere:
- **Hype Slider**: Increasing hype swells crowd noise and high-passes the main beat.
- **The Drop**: Slam the slider to zero to bring the bass back for maximum impact.

### 4. ✨ Gemini AI Integration
- **Text-to-Beat**: Generate drum patterns from text descriptions (e.g., "dark gritty techno").
- **AI DJ Festival Set**: Upload MP3s and let the AI build a setlist with automated FX.
- **Roast My Beat**: Get savage feedback from a "Berlin Techno Producer" AI.

## 🛠 Tech Stack
- **Web Audio API**: High-fidelity synthesized drums and real-time FX.
- **Vanilla JavaScript/CSS**: Ultra-fast performance and no dependencies.
- **Google Gemini API**: Advanced music analysis and generation.

## 🔑 Setup
To use the AI features, add your Google Gemini API Key to the `apiKey` variable in `index.html`.
